# Spring Boot Projesi

Bu proje, Java Spring Boot kullanılarak geliştirilmiş temel bir backend uygulamasıdır.

## Kullanılan Teknolojiler
- Java
- Spring Boot
- Maven
- REST API
- IntelliJ IDEA

## Proje İçeriği
- Katmanlı mimari (Controller, Service, Repository)
- Temel CRUD işlemleri
- RESTful servis yapısı
- Nesne yönelimli programlama (OOP) prensipleri

## Amaç
Spring Boot framework’ünü öğrenmek ve backend geliştirme mantığını kavramak amacıyla geliştirilmiştir.

## Çalıştırma
- Proje Maven ile derlenmiştir
- IDE üzerinden `Run` edilerek çalıştırılabilir
